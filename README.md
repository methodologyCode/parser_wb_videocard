# Парсер видеокарт.

```
Функционал:
- Парсит информацию о видеокартах определенных брендов.
- Можно указать определенный бренд, видеокарты которого нужно спарсить.
- Можно расширить кол-во брендов.
- Можно указать лимит.
- Можно указать кол-во страниц, которые нужно спарсить.
- Сохранение в формате CSV.
- Валидация json с помощью Pydantic.
```
### Стек
```
Python 3.7, requests, pydantic.
```
